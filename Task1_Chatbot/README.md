# AI Chatbot

## Description
This is a simple AI Chatbot developed using Python as part of the CodSoft AI Internship. The chatbot interacts with users by responding to predefined questions and ends the conversation when the user types 'bye', 'exit', or 'quit'.

## Features
- Greets the user
- Responds to common questions
- Accepts user input
- Ends the chat on command
- Easy to understand and modify

## Technologies Used
- Python 3
- Visual Studio Code

## How to Run
1. Open the project folder in Visual Studio Code.
2. Open the terminal.
3. Run the following command:
   ```
   python chatbot.py
   ```
4. Start chatting with the bot.
